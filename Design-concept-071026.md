Core Functionalities & Built-In Magic Features
1. Multi-Agent Orchestration Visualizer
Interactive Node Map: Implemented a futuristic live agent map visualizing the active states of the four core agents: Retrieval Agent (Web-RAG), Writing Agent (Dossier Compiler), Auditor Agent (Consistency Fact-Checking), and Temporal Agent (ISO 8601 timeline parsing).
Telemetry Stream: Feeds scrolling, timestamped terminal-style logs showing detailed agent actions, warning alerts, and success signals alongside a smooth progress loading gauge.
2. Dual-Pane Synchronized Markdown Editor
High-Fidelity Twin View: Real-time synchronized layout showing raw Markdown editing on the left and a pixel-perfect, CSS-styled document rendering on the right.
In-Place AI Command Toolbar: Selecting text inside the editor triggers a floating AI popover, allowing users to ask Gemini to rewrite, expand, or format the highlighted paragraph.
Corporate Export Controls: Integrated one-click downloads for raw Markdown documents, and a Print/Export to PDF button styled to prevent border clipping or pagination splits on cross-border table structures.
3. Executive Analytics Dashboard (6 "Wow" Visualizations)
Designed six high-fidelity SVG graphics rendering dynamic regulatory performance metrics:
Regulatory Compliance Ring: Displays multi-jurisdiction compliance percentage targets.
Risk Radar Profile: Identifies legal exposure under cybersecurity, audit, and label categories.
Timeline Gaps Analysis: Plots milestone timelines.
Adversarial Defense Score: Evaluates defense readiness.
Term Semantic Match Heatmap: Charts synonym translation accuracy.
Step-by-Step API Execution Cost: Compiles token usage logs.
4. Playgrounds for the 6 Core + 3 Additional Wow AI Features
Regulatory Risk Radar: Interactive database highlighting global penalties (e.g., EU MDR fines up to €30,000,000) and actionable mitigation checklists.
Ontology Term Alignment Matrix: Multi-lingual spreadsheet horizontally aligning equivalent legal concepts (e.g., Legal Responsible Entity to US, EU, and Taiwan TFDA definitions).
Timeline Gantt Milestone Generator: Visualizes scattered regulation deadlines as chronological milestones.
Regulatory Lineage Tracer: Clickable version trees depicting evolution pathways, such as MDD directives transitioning to amended MDR frameworks.
Compliance Gap Auditor with One-Click AI Autofill: Scans documents for missing subsections. Clicking One-Click AI Autofill calls our server proxy to generate and append compliant text to the document automatically.
Dual-Ended Engineering Change Impact Simulator: Engineers input spec changes (e.g., upgrading firmware and Bluetooth components). The AI assesses the re-registration thresholds across the US FDA, EU MDR, and TFDA.
Micro-Parameter Widget: Slides physical/clinical parameters (Risk Class, Body Contact, AI inclusion, Reusable) to immediately recalculate deadlines and test requirements.
Adversarial Auditor Q&A Simulator: Evaluates responses to strict deficiency letters from FDA/EMA reviewers, grades them out of 100, and provides perfect defense rationale paragraphs.
Five-Country Comparing Matrix: Highly structured, dense grid detailing operators, labeling, and cybersecurity rules for the US, EU, Taiwan, Brazil, and Australia.
