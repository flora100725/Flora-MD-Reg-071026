Functional Modules
1. Ingestion, Advanced Filter & Smart Normalizer
Smart Column Normalizer: Converts irregular raw text or CSV tables into strict TFDA schemas via a side-by-side comparative modal.
Traceability Filter Matrix: Implements instantaneous, multi-level filtration matching UDI-DI, sub-categories, temporal ranges, suppliers, and model numbers.
Taiwan Hub GIS Spatial Map: An interactive, vector-based SVG map detailing real-time shipment lanes and threat levels across 5 medical coordinates. Investigators can hover over coordinates for real-time metadata or click nodes to automatically lock filtration.
2. Dual-Pane Agentic Playground
ADK Simulator: Emulates the Google ADK execution threads, visualising active searches (google_search) and sandboxed calculation outputs (built_in_code_execution) to prevent manual recalculation of supply rates.
Workflow State Machine: Renders step-by-step progress counters through strategies like STRATEGY_PLANNING and CRITIQUE_LOOP in real time.
Report Compiler: Compiles clean Markdown reports with a single click, featuring a manual copy-paste context pane and raw file downloaders.
3. Proportional 6-Graph Analytic Dashboard
Ingestion & Delivery Trends: Grouped bar charts depicting temporal volumes for March/April 2026.
UDI/DI Product Category Breakdown: High-contrast donut/pie charts grouping pacemakers versus pulse generators.
Expiry Threat Timeline: Horizontal bar-charts showing serial codes closest to their expiration thresholds, colored by risk severity.
Compliance Deviation Scatter: Scatter map showing the volume of records versus active anomaly rates for each hub.
Supply Chain Network: Dynamic node flowchart visually tracking shipment lineages from border suppliers down to target hospitals.
Threat Radar Overlay: Polar radar tracking anomalies to isolate system-wide outliers.
4. Wow AI Interactive Features
Forensic Debate Engine (Adversarial Audit): Simulates structured adversarial arguments between the Auditor Agent and Defense Agent, synthesizing a final shared consensus audit report.
Regulatory Self-Critique Overlay: Automatically reviews the report against TFDA guidelines, flagging missing statutory citations in neon yellow with interactive Apply Correction buttons that repair the text in real-time.
